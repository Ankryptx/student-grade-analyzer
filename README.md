## 📌 What This Project Does
- 📂 Loads student marks from a CSV file
- 📊 Shows statistics and **visual charts**
- 🤖 Trains an **AI model** (Logistic Regression) to predict Pass/Fail
- 🔮 Takes new student marks and gives an **instant AI prediction**

## 🛠️ Tech Used
| Library | Purpose |
|---------|---------|
| Python 3 | Core language |
| Pandas | Data loading & analysis |
| Matplotlib | Charts & visualization |
| Scikit-learn | AI/ML model |

## 🚀 How to Run

### 1. Clone the repo
```bash
git clone https://github.com/ankrypt/student-grade-analyzer.git
cd student-grade-analyzer
```

### 2. Install requirements
```bash
pip install -r requirements.txt
```

### 3. Run the project
```bash
python analyzer.py
```
